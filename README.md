# yt-time (YouTube視聴時間制限拡張機能)

YouTubeの動画視聴時間を1日2時間に制限し、時間を超えた場合にその日のYouTubeの視聴を制限（ブロック）する軽量でセキュアなGoogle Chrome拡張機能（Manifest V3）です。

## 主な特徴

- **自動視聴時間測定**: YouTube動画が再生中の時間を秒単位で正確にカウントします。
- **マルチタブ対応**: 複数のタブで同時にYouTubeを開いて再生している場合でも、時間が重複して余分にカウントされないよう自動調整します。
- **日付変更時の自動リセット**: 毎日ローカル時間の深夜に、累積視聴時間が自動的にリセットされます。
- **プレミアムなブロック画面**: 制限時間を超えると、動画の再生を即座に強制停止し、スタイリッシュなアニメーション付きのブロック画面を表示して操作を無効化します。
- **安全第一の実装 (XSS対策)**: `innerHTML` などの動的HTML挿入を完全に排除し、安全なDOM APIのみを使用して要素を生成しています。
- **軽量・高速**: 外部ライブラリに一切依存しないバニラJavaScript & CSS実装のため、ブラウザやYouTubeの動作を重くしません。
- **最小限の権限**: 拡張機能の動作に必要な最小限のパーミッション（`storage`, `tabs`）のみを使用しています。

## インストール方法

1. このリポジトリをクローンするか、ファイルをダウンロードします。
2. Google Chromeを開き、アドレスバーに `chrome://extensions/` と入力して移動します。
3. 拡張機能画面の右上にある「**デベロッパー モード**」のスイッチをオンにします。
4. 左上に表示される「**パッケージ化されていない拡張機能を読み込む**」ボタンをクリックします。
5. 本プロジェクト内の **`yt-time-extension`** フォルダを選択して読み込みます。

## 使用方法と動作確認（テスト方法）

### 通常の使用方法
- 拡張機能をロードすると、YouTubeで動画を再生している間のみ、自動的にバックグラウンドで視聴時間がカウントされます。
- ブラウザのアクションバーから拡張機能のアイコンをクリックすると、現在の「本日の視聴時間」、「制限時間（デフォルト：2時間）」、「進捗状況（プログレスバー）」、「ステータス」を確認できます。
- 視聴時間が2時間に達すると、YouTubeの画面全体が「制限時間に達しました」というメッセージで覆われ、動画再生が停止します。

### 動作テスト（デバッグ機能）
2時間待つことなく動作を確認できるよう、ポップアップにテスト用ツールを搭載しています。

1. 拡張機能のポップアップを開きます。
2. ポップアップ下部の「デバッグツール」内にある「**テスト用制限（10秒）**」のトグルスイッチをオンにします。
3. YouTubeで任意の動画を再生し、10秒が経過すると即座に画面全体がブロックされることを確認してください。
4. 再び視聴できるようにするには、ポップアップの「**視聴時間をリセット**」ボタンをクリックします。
